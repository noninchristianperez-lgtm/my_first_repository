LENGUAJES DE PROGRAMACIÓN

1. Concepto

Un lenguaje de programación es un sistema formal de notación que permite escribir instrucciones que una computadora puede ejecutar. Es el medio de comunicación entre el programador y la máquina.

Permite traducir ideas lógicas en código que el computador interpreta y ejecuta, haciendo posible crear software, aplicaciones y páginas web.

Cada lenguaje tiene una sintaxis (reglas de escritura) y una semántica (significado de cada instrucción).

2. Evolución Histórica

1843

Ada Lovelace

Primer algoritmo para máquina analítica. Primera programadora de la historia.

1940-50

Leng. de 1ra Gen.

Código máquina y ensamblador. Programación directa en binario.

1957

FORTRAN

Primer lenguaje de alto nivel. Revolución en programación científica.

1970

C

Base de UNIX y Linux. Control directo del hardware.

1991

Python

Creado por Guido van Rossum. Popular para IA y data science.

1995

Java

"Escribir una vez, ejecutar en cualquier lugar". Base de Android.

1995

JavaScript

El lenguaje esencial de la web. Páginas interactivas.

2010s

Era Moderna

Swift, Kotlin, Rust, TypeScript. Enfoque en seguridad.

3. Clasificación por Nivel

Bajo Nivel

Más cercanos a la máquina. Requieren conocimiento del hardware.

Código Máquina, Ensamblador

Nivel Medio

Entre hardware y software. Combinan control y abstracción.

C, C++, Rust

Alto Nivel

Cercanos al humano. Fáciles de aprender y usar.

Python, JavaScript, Java, Ruby

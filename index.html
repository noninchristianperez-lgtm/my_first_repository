TIPOS DE
HARDWARE

Anatomía funcional de los sistemas informáticos modernos bajo estándares profesionales.

// INPUT_LAYER

Hardware de Entrada

Periféricos encargados de la captura de datos externos para ser transformados en señales digitales procesables.

⌨️
Teclado

Introduce datos alfanuméricos y comandos mediante el cierre de circuitos en una matriz de interruptores.

🖱️
Mouse

Traduce el movimiento físico o detección óptica en vectores de posición sobre la interfaz gráfica.

🎙️
Micrófono

Convierte ondas mecánicas sonoras en impulsos eléctricos para la digitalización de audio.

📄
Escáner

Analiza ópticamente imágenes o textos físicos para convertirlos en representaciones de mapas de bits.

📷
Webcam

Sensor de imagen digital que captura fotogramas secuenciales para transmisiones de video en vivo.

// OUTPUT_LAYER

Hardware de Salida

Dispositivos que transforman los resultados binarios del procesamiento en señales perceptibles por el usuario.

🖥️
Monitor

Genera una representación visual mediante la activación de subpíxeles en paneles LED, OLED o IPS.

🔊
Altavoces

Transducen señales eléctricas analógicas en energía acústica para la reproducción de sonido.

🖨️
Impresora

Plasma datos digitales en sustratos físicos utilizando tecnologías de inyección térmica o tóner láser.

🎧
Auriculares

Ofrece una salida de audio personalizada con aislamiento acústico mediante controladores dinámicos.

📽️
Proyector

Amplifica y proyecta haces de luz modulados para mostrar imágenes en superficies de gran escala.

// DATA_PERSISTENCE

Hardware de Almacenamiento

Medios físicos diseñados para retener información de forma persistente, garantizando la integridad de los datos sin energía.

⚡
SSD NVMe

Utiliza chips de memoria NAND Flash para accesos de datos casi instantáneos sin partes móviles.

💿
HDD

Almacenamiento masivo que emplea platos magnéticos y cabezales mecánicos para guardar bits de datos.

🔌
USB Flash

Dispositivo de almacenamiento extraíble que implementa el protocolo USB para transporte de archivos.

🎴
Tarjeta SD

Memoria flash de alta densidad optimizada para dispositivos portátiles y captura de medios.

📀
Blu-ray

Soporte de lectura/escritura óptica que usa láser violeta para comprimir datos en alta definición.

// COMPUTATION_CORE

Hardware de Procesamiento

Componentes que conforman la unidad lógica central, ejecutando algoritmos y gestionando el flujo de instrucciones.

🧠
CPU

Cerebro del sistema que procesa todas las operaciones aritméticas y lógicas de los programas.

🎮
GPU

Procesador especializado en la aceleración de gráficos y cálculos matemáticos paralelos.

📚
Memoria RAM

Almacén de datos volátil de alta velocidad que sirve como espacio de trabajo para la CPU.

🔌
Placa Base

El hub central donde se conectan e intercambian datos todos los componentes electrónicos.

🔋
Fuente de Poder

Convierte y regula la tensión de la red eléctrica en los voltajes DC específicos del hardware.

// NETWORKING_STACK

Hardware de Comunicación

Interfaces y dispositivos encargados de la interconexión, transmisión y recepción de datos entre sistemas remotos.

📡
Router

Gestiona y enruta los paquetes de datos entre diferentes redes, asegurando la entrega al destino correcto.

🛜
Tarjeta Wi-Fi

Módulo que permite la transmisión de datos mediante ondas de radio siguiendo los estándares IEEE 802.11.

🌐
Módem

Modula y demodula señales para permitir la conexión de datos a través de líneas telefónicas o fibra óptica.

🔀
Switch

Conecta múltiples dispositivos en una red local (LAN) dirigiendo los datos solo al puerto del destinatario.

📲
Adaptador Bluetooth

Facilita la comunicación inalámbrica de corto alcance para el intercambio de datos entre periféricos.

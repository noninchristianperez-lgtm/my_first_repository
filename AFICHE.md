# Afiche Web: Lenguajes de Programación

## Concepto y Visión
Un afiche digital interactivo que presenta de forma visual y atractiva la información sobre Lenguajes de Programación. Diseño moderno tipo póster editorial con secciones claras, gráficos CSS decorativos y animaciones sutiles que guían al lector por el contenido.

## Lenguaje de Diseño

### Dirección Estética
Estilo editorial/moderno inspirado en infografías de revistas tecnológicas. Uso de tipografía bold, espacios amplios y acentos de color para hiérarchizar la información.

### Paleta de Colores
- **Primary**: #6366F1 (Indigo vibrante)
- **Secondary**: #10B981 (Esmeralda)
- **Accent**: #F59E0B (Ámbar)
- **Background**: #0F172A (Slate oscuro)
- **Surface**: #1E293B (Slate medio)
- **Text Primary**: #F8FAFC (Blanco cálido)
- **Text Secondary**: #94A3B8 (Gris azulado)

### Tipografía
- Títulos: Inter (bold/extrabold)
- Cuerpo: Inter (regular/medium)

### Sistema de Espaciado
- Secciones: padding generoso 80px vertical
- Cards: padding interno 24px, gap 16px
- Máximo ancho de contenido: 1200px

### Filosofía de Movimiento
- Fade-in al scroll para cada sección
- Hover sutil en elementos interactivos (scale 1.02)
- Transiciones suaves de 300ms

## Estructura de Página

### Hero Section
- Título principal "LENGUAJES DE PROGRAMACIÓN"
- Subtítulo breve
- Decoración geométrica CSS

### Sección 1: ¿Qué es un Lenguaje de Programación?
- Definición clara en texto destacado
- Icono decorativo

### Sección 2: Historia y Evolución
- Timeline visual desde lenguaje máquina hasta lenguajes de alto nivel
- Fechas y descripciones

### Sección 3: Niveles de Lenguajes
- Tres cards con los niveles (Bajo, Medio, Alto)
- Ejemplos de cada nivel
- Iconos distintivos

### Sección 4: Lenguajes Más Usados en 2024
- Cards para Python, C/C++, JavaScript, Java, C#
- Descripción breve y área de uso

### Footer
- Fuente/credits

## Características
- Responsive (mobile-first)
- Animaciones al scroll
- Navegación sticky
- Diseño single-page tipo póster

## Tecnologías
- HTML5 + CSS3 (sin frameworks)
- Vanilla JavaScript
- Google Fonts (Inter)
